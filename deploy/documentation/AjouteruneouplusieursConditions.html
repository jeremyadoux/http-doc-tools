Ajouter une ou plusieurs "Conditions"

Utilisation de "Integration" ››



Les conditions sont à sélectionner parmi la liste proposée. Ces conditions peuvent être fonction de :

  • "Propriété" - existante ou non - égale ou différente à une certaine valeur etc ...
  • "SOLIDWORKS" - éventuellement par rapport au niveau de version de SOLIDWORKS
  • "Configuration" - existence d'une configuration particulière - nombre de configurations présentes dans le document etc ...
  • "Document" - si le document est une pièce, un assemblage ou une mise en plan - si le document comporte des équations - si le nom contient N caractères etc ...
  • "Pièce" - si la pièce contient N corps volumiques, surfaciques - si la pièce est un mécano-soudé etc ...
  • "Assemblage" - si l'assemblage contient N pièces - N niveau(x) - un composant particulier etc ...
  • "Plan" - si la mise en plan contient N feuille(s) - si la feuille N est un format particulier - si la mise en plan référence un document particulier etc ...


Les conditions peuvent être glissées :

  • soit dans le dossier "Et" - dans ce cas toutes les conditions doivent être respectées pour appliquer l'opération
  • soit dans le dossier "Ou" - dans ce cas toutes les conditions ne sont pas nécessaire d'être respectées


Toutes les conditions détaillées ci-après peuvent être déposées soit dans le dossier "Et" soit dans le dossier "Ou".

Lorsque qu'une condition est sélectionnée dans la liste des conditions, un icône vous indique pour quel(s) document(s) elle sera prise en compte :

Indique que la condition est possible pour les documents Pièce, Assemblage et Mise en plan.

Indique que la condition est possible pour les documents Pièce, Assemblage.

Indique que la condition est possible uniquement pour les documents Mises en plan.

Indique que la condition est possible uniquement pour les documents Pièces.

Indique que la condition est possible uniquement pour les documents Assemblages.

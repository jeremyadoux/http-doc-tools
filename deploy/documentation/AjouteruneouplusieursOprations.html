Ajouter une ou plusieurs "Opérations"

Integration ›› Utilisation de "Integration" ››


Les opérations sont à sélectionner parmi la liste proposée. Ces opérations peuvent être fonction de :

  • "Propriété" - efface une propriété - ajoute ou modifie une propriété - modifie un champs dans une valeur - ajoute une date etc ...
  • "Pièce" - change le matériau, la couleur - exporte le fichier etc ...
  • "Assemblage" - remplace le composant - Résoudre/Alléger un composant - Fixer/Libérer un composant etc ...
  • "Plan" - cache/affiche un calque - supprime les table - insère une table de révisions - insert une nomenclature - remplace les fonds de plan etc ...
  • "Configuration" - renomme une configuration - active/supprime une configuration - reconstruction d'une configuration etc ...
  • "Document" - zoom tout - reconstruction et barre de reprise - gestion de l'ombre, de la perspective, du RealView etc ...
  • "Divers" - créer un rapport de contrôle - placer une vignette, un titre (nom du document analysé) - lancement d'une macro etc ...
  • La définition d'une opération se fait par un simple Glisser/Déposer.


Toutes les opérations détaillées ci-après peuvent être déposées dans le dossier "Opérations".

Lorsque qu'une opération est sélectionnée dans la liste des opérations, un icône vous indique pour quel(s) document(s) elle sera prise en compte :

Indique que l'opération est possible pour les documents Pièce, Assemblage et Mise en plan.

Indique que l'opération est possible pour les documents Pièce, Assemblage.

Indique que l'opération est possible uniquement pour les documents Mises en plan.

Indique que l'opération est possible uniquement pour les documents Pièces.

Indique que l'opération est possible uniquement pour les documents Assemblages.

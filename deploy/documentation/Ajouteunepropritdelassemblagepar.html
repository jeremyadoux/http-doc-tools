Ajoute une propriété de l'assemblage parent

Utilisation de "Integration" ›› Détail des opérations ›› Propriété ››


Opération possible sur les pièces ou les assemblages.

Cette opération crée une propriété sur le document 3D (pièce ou assemblage) depuis une propriété de l'assemblage parent. Si la propriété existe déjà elle sera modifiée.

Exemple :

Créer dans la pièce ou l'assemblage la propriété "Designation" en récupérant la propriété "Description" présente dans l'assemblage parent.

Ajoute une propriété sur toutes les dépendances d'un document

Utilisation de "Integration" ›› Détail des opérations ›› Propriété ››


Opération possible sur les pièces, les assemblages ou les mises en plan.

Cette opération permet de créer ou de modifier une propriété dans toutes les dépendances du document.

Exemple :

Créer la propriété "Num_Projet", avec la valeur "255", dans toutes les dépendances du document sélectionné.

Si un assemblage est sélectionné : ajoute la propriété "Num_Projet" dans toutes les pièces et les sous-assemblages référencés dans cet assemblage.

Si une pièce est sélectionnée : ajoute la propriété "Num_Projet" dans toutes les dépendances (pièce symétrique, pièce dérivée ...).

Si une mise en plan est sélectionnée : ajoute la propriété "Num_Projet" dans toutes les dépendances de cette mise en plan.

  • On peut ajouter "|1" à la fin de la propriété pour ne modifier que le 1er niveau de dépendance.

  • Vous pouvez également utiliser les instructions codées pour spécifier la valeur de remplacement.

BIEN RESPECTER LA CASSE


<FileName> pour afficher le nom du document.

<PathName> pour afficher le chemin complet de sauvegarde du document.

$Nom de la propriété pour afficher une valeur de propriété du document (si le nom de la propriété contient un espace, utiliser %20% pour remplacer cet espace).

<Configuration> pour afficher le nom de la configuration active du document.

<Date> pour afficher la date.

<Time> pour afficher l'heure.

<NewLine> pour passer à la ligne.

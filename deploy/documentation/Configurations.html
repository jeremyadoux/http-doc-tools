Dans les documents SOLIDWORKS les propriétés peuvent être affectées au document et/ou à des configurations spécifiques.

Dans cette fenêtre d'options nous définissons dans quel contexte seront lues ou appliquées les propriétés.



  • "Lecture des propriétés" :

Par exemple, on testera les propriétés sur la configuration "Dégagement", si elle existe.

La lecture des propriétés peut être réalisé sur la configuration active et sur le document si la propriété est absente de la configuration active.


  • "Application des propriétés" :

Par exemple, on appliquera les propriétés sur la configuration "Position ouverte", si elle existe.


Cela peut donc s'avérer très pratique pour récupérer les propriétés d'une configuration et les basculer sur une autre.

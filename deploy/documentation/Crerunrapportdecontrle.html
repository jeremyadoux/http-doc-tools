Créer un rapport de contrôle

Integration ›› Utilisation de "Integration" ›› Détail des opérations ›› Divers ››


Opération possible sur les pièces, les assemblages ou les mises en plan.

Cette opération permet de créer un rapport de contrôle. Le fichier peut être simplement au format texte (.txt) ou au format Word (.doc).

La sélection du nom du rapport peut se faire en cliquant sur si le fichier existe.

Si le fichier n'existe pas le nom peut être spécifié dans le champs "Nom du rapport à créer".

Vous pouvez utiliser les codes suivants pour nommer le rapport, le symbole "&" pour cumuler les champs est facultatif :

<User> affiche le nom de l'utilisateur.

<Date> affiche la date de génération du rapport.

<Time> affiche l'heure de génération du rapport.

<Temp> affiche le répertoire temporaire.

<Desktop> affiche le répertoire du bureau.

<UserProfile> affiche le répertoire de l'utilisateur.

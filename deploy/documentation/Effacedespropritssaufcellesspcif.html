Efface des propriétés sauf celle(s) spécifiée(s)

Utilisation de "Integration" ›› Détail des opérations ›› Propriété ››


Opération possible sur les pièces, les assemblages ou les mises en plan.

Cette opération permet de supprimer, du document parcouru, toutes les propriétés sauf certaines.

Exemple : Suppression de toutes les propriétés sauf la propriété "Traitement".

Vous pouvez utiliser le caractère "*" placé devant ou derrière une chaîne de caractères spécifique, afin de tester l'existence de propriété(s) dans le document, comportant la chaîne de caractères.

Exemple :

Pro* suppression de toutes les propriétés sauf celle(s) commençant par Pro dans le document.

*000 suppression de toutes les propriétés sauf celle(s) se terminant par 000 dans le document.

Vous pouvez également définir plusieurs propriétés, à éviter de supprimer, en les séparant par le caractère ";".

Exemple :

suppression de toutes les propriétés sauf Code_Article, Traitement et Fournisseur dans le document.

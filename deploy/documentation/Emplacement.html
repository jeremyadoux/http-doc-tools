Cet espace est réservé à la spécification des répertoires de recherches des documents, des références ou des fichiers.

 

 


  • Pour sélectionner un répertoire existant vous pouvez utiliser le bouton .
  • Pour ajouter le répertoire sélectionné il faut appuyer sur . Le répertoire s'ajoute à la liste.
  • En appuyant sur vous récupérer le dossier du document traité. Cette instruction est matérialisée par "<Document>" dans le chemin.
  • En appuyant sur vous spécifiez de récupérer le niveau au-dessus du répertoire où se trouve le document traité.
  • Si un répertoire doit être supprimé de la liste il suffit de le sélectionner et d'appuyer sur le bouton .


En cochant cette option vous autorisez la recherche dans tous les sous-dossiers.

Cette option peut ralentir considérablement les traitements.

Si cette option n'est pas cochée la recherche s'effectuera sur le nom du document uniquement.

Options d'export au format eDrawings:


Sauvegarder toutes les configurations ou feuilles: Permet d'exporter toutes les feuilles ou configurations dans le fichier eDrawings généré

Ne sauvegarder que la configuration ou feuille active: Permet de n'exporter que la feuille active ou configuration active même si le document est composé de plusieurs feuille ou configuration



Options d'export des mises en plans à feuille multiple:


Permet de sélectionner quelles feuilles exporter dans le cas de fichier PDF ou DXF par exemple.


L'option permet d'exporter la feuille vers l'espace papier (Autocad) dans le cas d'export au format DXF\DWG.

Exporter au format Excel les tables des plans

Utilisation de "Integration" ›› Détail des opérations ›› Plan ››


Opération possible sur les mises en plan.

Cette opération d'exporter, au format Excel, les tables spécifiées des plans vers un répertoire spécifique.


Il possible de choisir le type de table à exporter depuis un menu déroulant:


Exemple : Toute les nomenclatures dans le répertoire  "D:\Samples\180-Integration\Exports" seront exportées au format Excel.

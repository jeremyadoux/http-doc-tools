Exporter le fichier

Utilisation de "Integration" ›› Détail des opérations ›› Assemblage ››


Opération possible sur les pièces.

Cette opération permet d'exporter tous les fichiers sélectionnés avec un format d'export dans un dossier spécifique.

Le format d'export est choisi parmi la liste déroulante.

Elle correspond à tous les formats disponibles de la fonction "Enregistrer sous" de SOLIDWORKS.

Exemple :

Exporter, au format IGES, tous les fichiers sélectionnés.

Sélection 1 : "Spécifier un dossier ci-dessus pour les fichiers exportés"

Les exports seront sauvegardés dans le répertoire "D:\Samples\180-Integration\Exports".

Dans ce cas on spécifie un dossier sans tenir compte des options de renommage.

Sélection 2 : "Utiliser les options du programme ‘Option/Règles de nommage"

Les exports seront sauvegardés dans le répertoire "D:\TEMP\IGES" spécifié dans les règles de nommage.

Avec cette solution on ne spécifie pas de dossier d'export, l'emplacement sera défini dans les options de l'utilitaire (Règles de nommage), ainsi que les règles de nommage des fichiers.


Cette opération peut être précédée par d'autres opérations telles que :

- Effectuer un plein zoom

- Mettre le document en vue isométrique

- ...

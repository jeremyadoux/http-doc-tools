Gestion de la barre de blocage

Integration ›› Utilisation de "Integration" ›› Détail des opérations ›› Pièce ››


Opération possible sur les pièces.

Cette opération permet de gérer la barre de blocage sur les fichiers pièces.


- Gérer la position :


Descendre la barre de blocage en bas de l'arbre de création ou la désactiver (Barre de blocage en haut de l'arbre de création).



- Mise à jour sur quelle configuration :


Mettre à jour la configuration active ou toutes les configurations.

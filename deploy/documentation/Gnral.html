Le fait de cocher cette option permet de reprendre le dernier projet  ""(ou liste de conditions et d'opérations) à l'ouverture de l'utilitaire "Integration".

On spécifie ici le répertoire dans lequel seront enregistrés, par défaut, les projets "". Le choix du répertoire se fait en cliquant sur .

Lorsqu'un projet est ouvert avec il sera ajouté à la liste des critères de modification déjà présente.

Lorsqu’un traitement (projet) est lancé vous pouvez forcer la fermeture de tous les fichiers SOLIDWORKS déjà ouverts. Si en plus la case "Redémarrer SOLIDWORKS" est cochée vous pouvez forcer la fermeture de SOLIDWORKS tous les "N" documents. Effectivement pour le traitement d'un grand nombre de fichiers cela permet de réinitialiser tous les paramètres SOLIDWORKS.

Le fait de cocher cette option permet, à chaque fermeture du document, de libérer le processus SOLIDWORKS de la mémoire. Indispensable pour le traitement d'un grand nombre de fichiers de taille importante.

La sélection du type de définition des couleurs impactera le choix fait dans condition couleur pièce ou opération changer couleur pièce.

Le fait de cocher cette option permet de récupérer, pour une propriété, soit le contenu "Valeurs évaluée" (ici Acier Allié) soit la "Valeur / Expression de texte" (ici "SW-Material@Pièce1.SLDPRT").

Cette option est cochée par défaut.

Le fait de cocher cette option ajoute systématiquement une propriété "SaveByIntegration" sur tous les documents traités.

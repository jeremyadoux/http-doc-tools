Généralités des utilitaires

Introduction ››




Tous les utilitaires de la barre d'outils myCADtools présentent un bandeau identique comportant certaines informations.


Dans la partie gauche du bandeau nous retrouvons le nom de l'utilitaire suivi par le numéro de la version installée (exemple v2013.11.12) :

  • 2013→ Version de myCADtools
  • 11→ Année du setup
  • 12→ Mois du setup


Dans la partie droite nous avons accès à une barre d'icônes.

Possibilité de "punaiser" la fenêtre de l'utilitaire afin qu'elle ne soit pas minimisée par Windows.


Cet icône vous donne accès directement au cours afin de visualiser les vidéos de formation de l'utilitaire

Cet icône vous donne accès directement au site Visiativ


Cet icône vous donne accès directement au site myCADservices

En cliquant sur cet icône vous accédez aux informations liées à l'utilitaire mais également à l'installation de l'ensemble des utilitaires du setup myCADtools.


   


Voici les informations disponibles dans cette fenêtre :


  • Un rappel de la version de l'utilitaire mais également la version du setup d'installation de la barre d'outils myCADtools.


  • Un rappel du code d'immatriculation ainsi que le numéro de licence SolidWorks à laquelle est rattaché ce code.


  • La date limite d'utilisation de l'utilitaire par exemple dans le cas d'une demande de prêt.


  • Le nom des utilitaires installés ainsi que, pour chacun d'entre eux, le nombre d'utilisations. Cette information peut être une bonne indication afin de connaître quels sont les utilitaires les plus souvent utilisés.


  • Une possibilité de remonter des demandes d'amélioration via votre messagerie. Un message est alors directement envoyé à notre service R&D. Vous n’avez plus que le contenu du message à rédiger.


Information pour l’exploitation des données d’un coffre-fort ePDM avec certains outils myCADtools.

Les utilitaires de la suite myCADtools ne sont pas développés pour être utilisés dans un environnement PDM. Pour bénéficier de cette fonctionnalité, il faut impérativement avoir :

  • Installé SolidWorks Enterprise PDM
  • Installé la suite d’outils myPDMtools
  • Une licence d’immatriculation myPDMtools

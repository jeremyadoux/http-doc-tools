Groupes de conditions dans le dossier "Ou"

Utilisation de "Integration" ›› Ajouter une ou plusieurs "Conditions" ››


On peut imbriquer les conditions dans le dossiers "Ou" afin de réaliser des groupes de conditions. En effet les conditions déposées sur une condition déjà présente dans le dossier "Ou" sera considérée comme une condition "Et".

Exemples :

- "Si une propriété existe" : "Code_Article" (faire glisser cette condition dans le dossier "Ou").

- "Si le nombre de caractères de la valeur" est égal à "4" (Faire glisser cette condition dans le dossier "Ou" sur la première condition).

- "Si une propriété existe" : "Finition" (faire glisser cette condition dans le dossier "Ou").

- "Si le nombre de caractères de la valeur" est égal à "3" (Faire glisser cette condition dans le dossier "Ou" sur la deuxième condition).


Ce qui donne :

Si la propriété "Code_Article" existe et que sa valeur est égale à 4 caractères.

Ou

Si la propriété "Finition" existe et que sa valeur est égale à 3 caractères.

Alors l'opération sera appliquée.

Incrémente une valeur de propriété avec les compteurs SmartCounter

Integration ›› Utilisation de "Integration" ›› Détail des opérations ›› Propriété ››


Opération possible sur les pièces, les assemblages ou les mises en plan.

Cette opération permet d'incrémenter la valeur d'une propriété en utilisant des compteurs.

Exemple :

La propriété "Compteur" suivra l'incrément déclaré dans le compteur "Incrementing0".



Le bouton permet de paramétrer les compteurs.


Voir le paramétrage des compteurs dans le paragraphe "SmartCounter".

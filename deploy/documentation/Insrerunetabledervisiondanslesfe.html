Insérer une table de révision dans les feuilles d'une mise en plan

Integration ›› Utilisation de "Integration" ›› Détail des opérations ›› Plan ››


Opération possible sur les mises en plan.

Cette opération insère une table de révision dans une ou plusieurs feuilles présentes dans la mise en plan.

Le numéro de la feuille est à choisir parmi une liste. Ce numéro doit être numérique. Il peut être saisi au clavier ou à l'aide de la liste déroulante :

Exemple : Une table de révision sera insérée sur la feuille "5" de toutes les mises en plan sélectionnées.

Le modèle de table de révision peut être sélectionné en cliquant sur .



Au-lieu de spécifier une feuille, vous pouvez utiliser le caractère "*" afin d'insérer une table de révision sur toutes les feuilles.


Si vous souhaitez insérer une table de révision sur plusieurs feuilles, utilisez le caractère ";" comme séparateur.

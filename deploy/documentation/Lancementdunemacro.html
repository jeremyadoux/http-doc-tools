Lancement d'une macro

Integration ›› Utilisation de "Integration" ›› Détail des opérations ›› Divers ››


Opération possible sur les pièces, les assemblages ou les mises en plan.

Cette opération permet d'exécuter une macro.

La macro peut être sélectionnée en cliquant sur .


Exemple : La macro "SuppressColor.swp" sera exécutée sur tous les documents sélectionnés.

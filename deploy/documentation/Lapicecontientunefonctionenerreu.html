Le document testé est une pièce.

Applique l'opération si une fonction est en erreur dans la pièce.

La valeur peut être en "Erreur" ou en "Alerte".

Exemple : Si la pièce contient au-moins une fonction en "Erreur", alors l'opération est appliquée.

"Erreur" : la fonction n'est pas construite.

"Alerte" : la fonction est construite avec une puce d'alerte.

Le champ "Nom de la configuration" est facultatif. Si rien n'est précisé cela veut dire que c'est la configuration active qui est testée. Utiliser le caractère "*" pour vérifier toutes les configurations de la pièce.

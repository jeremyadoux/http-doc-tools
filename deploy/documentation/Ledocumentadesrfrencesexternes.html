Le document testé est une pièce, un assemblage ou une mise en plan.


Vérification des références externes.

Vérifie si les emplacements sont valides (existence d'un document), avec la possibilité de rechercher les documents référencés :

- uniquement à l'emplacement enregistré dans le document

- à l'emplacement enregistré dans le document et fonction des règles de recherche spécifiées dans les options SOLIDWORKS.

Ensuite le niveau de recherche pour le contrôle peut s'arrêter au premier niveau ou descendre dans tous les niveaux.

Le document testé est une pièce, un assemblage ou une mise en plan.

Applique l'opération si le document contient N équations avec un état.

Le test est choisi parmi la liste "Égale à" ou "Différent de", "Supérieur ou égale à", "Supérieur à", "Inférieur ou égale à" ou "Inférieur à". Le nombre de caractères doit être numérique. Il peut être saisi au clavier ou à l'aide de la liste déroulante :

On choisira également, parmi une liste déroulante, l’état des équations "Toutes les équations", les équations dont l'état est "En erreur" ou les équations dont l'état est "Valide"

Exemple : Si le document contient "3" équations "En erreur" alors l'opération est appliquée.

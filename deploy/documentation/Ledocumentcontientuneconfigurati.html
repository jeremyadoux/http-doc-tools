Le document contient une configuration dont le nom est égal à une valeur donnée

Integration ›› Utilisation de "Integration" ›› Ajouter une ou plusieurs "Conditions" ›› Détail des conditions ›› Configuration ››


Le document testé est une pièce ou un assemblage.

Applique l'opération si le document parcouru contient une configuration "Égale à" ou "Différent de" la valeur spécifiée.

Exemple : Si, dans le document, le nom d'une configuration est égal à "Validée" alors l'opération est appliquée.

Vous pouvez utiliser le caractère "*" placé devant ou derrière une chaîne de caractères spécifique, afin de tester l'absence d'une certaine configuration dans le document, comportant la chaîne de caractères spécifiée.

Exemple :

100* si le nom d'une configuration commençant par la chaîne de caractères 100 existe dans le document, alors l'opération est appliquée.

*100 si le nom d'une configuration se terminant par la chaîne de caractères 100 existe dans le document, alors l'opération est appliquée.

 *100* si le nom d'une configuration contenant la chaîne de caractères 100 existe dans le document, alors l'opération est appliquée.

Le nom du document correspond à un format particulier

Le document testé est une pièce, un assemblage ou une mise en plan.

Applique l'opération seulement si le nom du document correspond à un format d'écriture particulier. On utilisera les caractères suivants pour définir la syntaxe :

$ = caractère alphanumérique

# = caractère numérique

? = caractère quelconque

* = remplace une chaîne de caractères

Exemple : Si le format du nom du document parcouru correspond à la syntaxe "###-$$-*" , alors l'opération est appliquée.

Ce format correspond à un nom de document de type : 568-DC-Plaque

Modifie un champ dans une valeur de propriété

Integration ›› Utilisation de "Integration" ›› Détail des opérations ›› Propriété ››


Opération possible sur les pièces, les assemblages ou les mises en plan.

Cette opération permet de modifier une partie de la valeur d'une propriété du document parcouru par une chaîne de caractères.

Exemple :

Dans la valeur de la propriété "Num_Projet", on souhaite remplacer "809" par "225". Utiliser le caractère "|" comme séparateur.


  • Vous pouvez utiliser le caractère "*" placé devant ou derrière une chaîne de caractères spécifique :
    • *123*|abc - Remplacer la chaîne de caractères 123 par abc quelque soit sa position dans le champ.

Exemple : La valeur 456-12345-APP deviendra 456-abc45-APP après application de l'opération.

    • 123*|abc - Remplacer la chaîne de caractères 123 par abc uniquement si elle est positionnée au début champ.

Exemple : La valeur 12345-APP-01 deviendra abc45-APP-01 après application de l'opération.

    • *123|abc - Remplacer la chaîne de caractères 123 par abc uniquement si elle est positionnée à la fin du champ.

Exemple : La valeur APP68-01-123 deviendra APP68-01-abc après application de l'opération.


  • Vous pouvez utiliser le caractère "#" placé devant ou derrière une chaîne de caractères spécifique :
    • #1s4|2013 - Remplacer par 2013 la zone du champ en commençant au 1er caractère sur 4 caractères.

Exemple : La valeur 4560-45-APP deviendra 2013-45-APP après application de l'opération.

    • #3e3|2013 - Remplacer par 2013 la zone du champ depuis le 3ième caractère en partant de la fin sur 3 caractères.

Exemple : La valeur PROJ-45-1999 deviendra PROJ-45-2013 après application de l'opération.

    • *123|abc - Remplacer la chaîne de caractères 123 par abc uniquement si elle est positionnée à la fin du champ.

Exemple : La valeur APP68-01-123 deviendra APP68-01-abc après application de l'opération.


  • Plutôt que de spécifier une chaîne de caractères dans la valeur de remplacement, vous pouvez utiliser le caractère "$" afin de récupérer le contenu d'une propriété du document :
    • #1s4|$Code_Article - Remplacer par la valeur de la propriété "Code_Article" la zone du champ en commençant au 1er caractère sur 4 caractères.

Exemple : La valeur 4560-45-APP deviendra 3-22-45-APP après application de l'opération en considérant que la propriété "Code_Article" est égale à "3-22".


  • Vous pouvez également utiliser les instructions codées pour spécifier la valeur de remplacement. BIEN RESPECTER LA CASSE

<FileName> pour afficher le nom du document.

<PathName> pour afficher le chemin complet de sauvegarde du document.

$Nom de la propriété pour afficher une valeur de propriété du document (si le nom de la propriété contient un espace, utiliser %20% pour remplacer cet espace).

<Configuration> pour afficher le nom de la configuration active du document.

<Date> pour afficher la date.

<Time> pour afficher l'heure.

<NewLine> pour passer à la ligne.

Format de compteur ’’A,B,C’’

Pas :

Spécifier le pas pour incrémenter le compteur.

Avec un pas de 1 le compteur s’incrémentera comme suit : A ; B ; C ; D etc.

Avec un pas de 2 le compteur s’incrémentera comme suit : A ; C ; E ; G etc.

    • Valeur de départ : Définir la valeur de départ.
    • Valeur : Permet de visualiser la dernière valeur appliquée sur un fichier.
    • Valeur suivante : Permet de visualiser la prochaine valeur qui sera appliquée sur un fichier.

Pas :

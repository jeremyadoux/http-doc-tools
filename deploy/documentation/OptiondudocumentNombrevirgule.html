Option du document (Nombre à virgule)

Utilisation de "Integration" ›› Détail des opérations ›› Document ››


Opération possible sur les pièces, les assemblages ou les mises en plan.

Cette opération permet d'accéder aux options type "Nombre à virgule" des documents. Pour plus d'information sur l'utilisation de ces paramètres, voir aide API (Aide/Aide API).

L'utilisation de cette opération est réservée aux utilisateurs avancés.

Le choix se fait parmi une liste d'option correspondant à l'énumérateur "swUserPreferenceDoubleValue_e" :

 

 

La valeur doit être un nombre (entier ou à virgule).

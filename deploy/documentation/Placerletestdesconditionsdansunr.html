Placer le test des conditions dans un rapport

Utilisation de "Integration" ›› Détail des opérations ›› Divers ››


Opération possible sur les pièces, les assemblages ou les mises en plan.

Cette opération permet le test des conditions, dans le rapport de contrôle.

Le rapport doit, au préalable, avoir été créé.

Le choix du rapport se fait en le choisissant parmi la liste des opérations ayant créés des rapports de contrôle.


Exemple : Le test du document traité sera "Toujours" rajouté dans le rapport "Diffusion.doc". (Voir un exemple de rapport).

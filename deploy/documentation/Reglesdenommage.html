Renommage

Cette rubrique permet d'établir les règles de nommage des documents qui seront traits dans les outils.

Pour ce faire nous disposons de 2 masques de réglages :

  • Le mode "Simplifié" qui limite le nombre à 4 règles à prendre en compte ou non.
  • Le mode "Avancé" qui permet de créer plusieurs règles, d'enregistrer et lire des favoris de règles prédéfinies etc ...

Pour chaque règle sélectionnée il est possible d'avoir un aperçu du renommage.

On peut définir un emplacement par défaut pour l'enregistrement du document après renommage.

L’ajout d'un champ pour certaines règles peut être renseigné d'une certaine manière.

Mode simplifié

Le mode simplifié propose 4 règles de nommage que l'on peut activer ou non. Les 3 premières peuvent être activées individuellement. L'activation de la quatrième désactive automatiquement les autres étant donné que le nom est complètement remplacé.



Dans l'exemple ci-dessus on a activé les 3 premières règles.

On remplace ou ajoute les 2 premiers caractères avec "AX" suivi de "Projet", ensuite on remplace les 2 derniers caractères du nom par "PR" et enfin on remplace la chaîne de caractères "245" par "180".

Voir l'aperçu du renommage.

Renseigner le champ soit manuellement soit cliquer sur ajouter un champ particulier.

Mode avancé

Cette rubrique permet d'établir les règles de nommage des documents qui seront traités dans le projet.

La définition du nouveau nom va suivre une "Liste des règles sélectionnées".

Le fait de cliquer sur pour ajouter un champ, permet d'accéder à une fenêtre afin de sélectionner un paramètre particulier.

Remplacer- Ajouter les premiers caractères

Permet de remplacer ou d'ajouter N caractères depuis le début.

La zone numérique permet de rogner N caractères en tête du nom.

Appuyer sur le bouton afin d'ajouter cette règle à la liste.

Exemple :

Pour un nom actuel "AX-2012-0095.sldprt", cette règle modifie le nom comme suit "PR-2012-0095.sldprt".


Si la valeur numérique est égale à zéro (0) les caractères seront insérés sans écraser le début du nom.

Exemple :

Pour un nom actuel "2012-0095.sldprt", cette règle modifie le nom comme suit "PR2012-0095.sldprt".

On peut paramétrer la règle en fonction de critères tels que : le type de fichier SolidWorks, la valeur d'une propriété etc ... (voir Liste des règles sélectionnées).

Voir l'aperçu du renommage.


Remplacer- Ajouter les derniers caractères

Permet de remplacer ou d'ajouter N caractères à la fin.

La zone numérique permet de rogner N caractères en fin de nom.

Appuyer sur le bouton afin d'ajouter cette règle à la liste.

Exemple :

Pour un nom actuel "AX-2012-0095-TU.sldprt" Cette règle modifie le nom comme suit "AX-2012-0095.sldprt".


Si la zone texte est renseignée, les caractères seront ajoutés à compter de la fin du nom.

Exemple :

Pour un nom actuel "AX-2012-0095-TUB.slddrw" Cette règle modifie le nom comme suit "AX-2012-0095-Plan.slddrw". Cette règle peut s'exécuter à condition que le document soit une mise en plan.

On peut paramétrer la règle en fonction de critères tels que : le type de fichier SolidWorks, la valeur d'une propriété etc ... (voir Liste des règles sélectionnées).

Voir l'aperçu du renommage.


Remplacer une partie du nom

Permet de remplacer une chaîne de caractères contenue dans le nom par une autre.

Appuyer sur le bouton afin d'ajouter cette règle à la liste.

La première zone de texte permet de définir la chaîne à remplacer.

La seconde zone de texte permet de définir la chaîne de remplacement.

Exemple :

Pour un nom actuel "AX-2012-0095.sldprt", cette règle modifie le nom comme suit "PR-2013-0095.sldprt".

Voir l'aperçu du renommage.


Suppression des caractères à partir de

Permet de supprimer un nombre de caractères depuis une position dans le nom.

Appuyer sur le bouton afin d'ajouter cette règle à la liste.

La première valeur numérique définit la position de départ.

La seconde valeur numérique définit la longueur de la chaîne de caractères à supprimer.

Exemple :

Pour un nom actuel "Axemble-2012-0095.sldprt", cette règle modifie le nom comme suit "Ax-2012-0095.sldprt".

Voir l'aperçu du renommage.


Insertion de caractères, à la position

Permet d’insérer une chaîne de caractères à partir d’une position dans le nom.

Appuyer sur le bouton afin d'ajouter cette règle à la liste.

La valeur numérique permet de définir la position de départ.

La zone de texte permet de définir la zone de texte à insérer.

Exemple :

Pour un nom actuel "AX-2013-0095 Usiné.sldprt", cette règle modifie le nom comme suit "PR-2013-0095 et usiné.sldprt".

Pour utiliser cette règle il est nécessaire que la position du texte à insérer soit toujours la même.

Voir l'aperçu du renommage.


Modifier la casse

Permet de modifier la casse du nom complet (minuscule/Majuscule/Première lettre en Majuscule).

Appuyer sur le bouton afin d'ajouter cette règle à la liste.

Exemple :

Pour un nom actuel "Ax-2013-0095 et Usiné.sldprt", cette règle modifie le nom comme suit "AX-2013-0095 ET usiné.sldprt".

Voir l'aperçu du renommage.


Supprimer les caractères suivants

Permet de supprimer une chaîne de caractères contenue dans le nom.

Appuyer sur le bouton afin d'ajouter cette règle à la liste.

Exemple :

Pour un nom actuel "Axemble-2013-0095.sldprt", cette règle modifie le nom comme suit "Ax-2013-0095.sldprt".

Cette chaîne de caractères sera supprimée dans tout le nom. C'est à dire que le nom "Axemble-2013-0095 Sous-Ensemble.sldprt" deviendra "Ax-2013-0095 Sous-Ens.sldprt".

Voir l'aperçu du renommage.

Ajout d’un champ

Pour renseigner le champ d'une règle, le fait de cliquer sur permet d'accéder à une fenêtre afin de sélectionner un paramètre particulier.

Les champs possibles sont :

- Sélectionner une valeur fixe (c'est ce que l'on a fait pour la première règle).

- Sélectionner la valeur d'une propriété (avec le choix parmi la liste)

- Sélectionner une valeur d'un compteur SmartCounter (Voir paragraphe ’’SmartCounter’’), une date, l'heure etc...


Liste des règles sélectionnées

Il est possible de modifier l'ordre des règles en la sélectionnant puis en appuyant sur les boutons monter ou descendre.

Vous pouvez sauvegarder un favori de liste de règles afin de les réutiliser par la suite. L'extension de ces fichiers est ".mcren".

La relecture d'un favori sauvegardé se fait par le bouton.

Toutes les règles de la liste peuvent être conditionnées par rapport à une information. En cliquant sur vous ouvrez un boîte de dialogue comme suit :



Il ne peut y avoir qu'une seule condition de cochée. En effet on ne peut pas cocher les 3 conditions suivantes en même temps :

- si le document est une mise en plan

- si la propriété "Révision" est égale à "B"

- si la taille du plan est "plus grande ou égale" à "1Mo"

Vous pouvez également accéder à cette boîte en cliquant sur la condition d'une règle de la liste puis en appuyant sur le bouton .

En cliquant sur une règle de la liste vous pouvez la désactiver temporairement (Activée) ou (Désactivée).

Aperçu du nouveau nom

En cliquant sur l'aperçu du nouveau nom est visible et modifié en direct.

Lorsque l'on crée une nouvelle règle, l'aperçu du nouveau nom se met à jour automatiquement. Dans ce cas l'icône n'est pas visible.


Emplacement du fichier

L'emplacement du fichier peut être défini comme étant le même répertoire que le document initial.

On peut néanmoins définir un emplacement spécifique. Pour ce faire l'interface propose 2 type de choix :

  • permet de sélectionner un répertoire via l'explorateur Windows.


  • permet de spécifier un chemin relatif du dossier. En partant du dossier du document on peut créer un dossier enfant dans lequel sera enregistré le document.

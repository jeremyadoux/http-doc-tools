Supprime la table de famille

Integration ›› Utilisation de "Integration" ›› Détail des opérations ›› Configuration ››


Opération possible sur les pièces ou les assemblages.

Cette opération permet de supprimer la table de famille du document, elle s'utilise avec la condition "En fonction de la table de famille" pour n'ouvrir que les documents comportant une table de famille (condition "Document").

Le document testé est une pièce, un assemblage ou une mise en plan.

Applique l'opération si une propriété existe dans le document.

Exemple : Si la propriété "Traitement" existe dans le document, alors l'opération est appliquée.

Vous pouvez utiliser le caractère "*" placé devant ou derrière une chaîne de caractères spécifique, afin de tester l'existence d'une propriété dans le document, comportant la chaîne de caractères.

Exemple :

Pro* si une propriété commençant par Pro existe dans le document, alors l'opération est appliquée.

*000 si une propriété se terminant par 000 existe dans le document, alors l'opération est appliquée.

Vous pouvez également définir plusieurs propriétés en les séparant par le caractère ";".

Exemple :

Si les propriétés Code_Article, Traitement et Fournisseur existent dans le document, alors l'opération est appliquée.

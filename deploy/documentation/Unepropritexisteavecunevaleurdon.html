Une propriété existe avec une valeur donnée

Integration ›› Utilisation de "Integration" ›› Ajouter une ou plusieurs "Conditions" ›› Détail des conditions ›› Propriété ››


Le document testé est une pièce, un assemblage ou une mise en plan.

Applique l'opération en fonction si une propriété, avec une certaine valeur, existe dans le document.

La valeur peut être "Égale à" ou "Différente de".

Exemple : Si la propriété "Matière" existe dans le document et que sa valeur est "Acier", alors l'opération est appliquée.

 

Vous pouvez également définir plusieurs valeurs possibles en les séparant par le caractère "§".

Exemple : Si la propriété "Matière" existe sur le document avec une des valeurs Acier, Alu ou Fonte, alors l'opération est appliquée.

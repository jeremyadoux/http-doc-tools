Utilisation de "Integration" ›› Ajouter une ou plusieurs "Conditions" ›› Détail des conditions ›› Configuration ››


Le document testé est une pièce ou un assemblage.

Applique l'opération si le nom de la configuration active est "Égale à" ou "Différent de" la valeur de la propriété.

Exemple : Si, dans le document, le nom de la configuration active est égal à "Validée" alors l'opération est appliquée.

Vous pouvez utiliser le caractère "*" placé devant ou derrière une chaîne de caractères spécifique, afin de tester l'absence d'une propriété dans le document, comportant la chaîne de caractères.

*100 si le nom de la configuration active se termine par la chaîne de caractères 100, alors l'opération est appliquée.

100* si le nom de la configuration active commence par la chaîne de caractères 100, alors l'opération est appliquée.

*100* si le nom de la configuration active contient la chaîne de caractères 100, alors l'opération est appliquée.

Menus contextuels de l'arbre du projet

Lorsque les actions ont été créées dans l'arbre, il est possible d'en apporter certaines modifications par le biais de menus contextuels.

En faisant un clique droit sur une action :

  • Vous avez la possibilité de renommer l'action :

  • Ou bien de dupliquer l'action. Cette dernière peut s'avérer utile si vous avez des actions très proches pour lesquelles une partie des actions et/ou des opérations est commune.

Cette dernière peut s'avérer utile si vous avez des actions très proches pour lesquelles une partie des actions et/ou des opérations est commune.

  • Chaque action comporte une fonction "Paramètres".

Par un clique droit sur cette fonction vous pouvez éditer les paramètres de l'action. Une fenêtre s'ouvre dans laquelle vous saisissez un commentaire de l'action.

Concernant la partie "Options", de cette fenêtre :

    • si la case est cochée l'action sauvegardera le document ouvert. En effet nous verrons plus loin que certaines conditions et opérations nécessitent l'ouverture ou non du document.
    • cette option est cochée par défaut. En général on applique l'opération en cas de succès de(s) condition(s). Si l'on décoche cette option, l'opération sera appliquée en cas d'échec.

Modifier le texte d'une note

Utilisation de "Integration" ›› Détail des opérations ›› Plan ››


Opération possible sur les mises en plan.

Cette opération permet de modifier une partie d'une note présente dans la mise en plan.


Exemple : Remplace le texte de la note "AF1523" par "NF1523" sur toutes les mises en plan sélectionnées.


Comme valeur de remplacement vous pouvez utiliser les instructions suivantes :


<FileName>, pour afficher le nom du document.

<PathName>, pour afficher le chemin complet du document.

$Nom de la propriété, pour afficher une valeur de propriété du document (si le nom de la propriété contient un espace, utiliser %20% pour remplacer cet espace).

<Configuration>, pour afficher le nom de la configuration active du moment.

<Date>, pour afficher la date.

<Time>, pour afficher l'heure.

<NewLine>, pour passer à la ligne

Une propriété n'existe pas

Utilisation de "Integration" ›› Ajouter une ou plusieurs "Conditions" ›› Détail des conditions ›› Propriété ››


Le document testé est une pièce, un assemblage ou une mise en plan.

Applique l'opération si une propriété n'existe pas dans le document.

Exemple : Si la propriété "Traitement" n'existe pas dans le document, alors l'opération est appliquée.

Vous pouvez utiliser le caractère "*" placé devant ou derrière une chaîne de caractères spécifique, afin de tester l'absence d'une propriété dans le document, comportant la chaîne de caractères.

Exemple :

Pro* si une propriété commençant par Pro n'existe pas dans le document, alors l'opération est appliquée.

*000 si une propriété se terminant par 000 n'existe pas dans le document, alors l'opération est appliquée.

Vous pouvez également définir plusieurs propriétés en les séparant par le caractère ";".

Exemple :

Si aucune des propriétés Code_Article, Traitement et Fournisseur n'existent dans le document, alors l'opération est appliquée.
